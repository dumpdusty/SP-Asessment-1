(inner) getSettlementDate(initialDate, delay)

Handles requests for calculating the date on which a settlement will reach a bank account, based on an initial date and a number of days of delay. This route handler requires a JSON payload with the parameters described below and will return a JSON object with `initialQuery` and `results` properties. The `initialQuery` property contains the data that was passed in the client request and `results` contains a hash that includes the `businessDate`, which is the actual date we're mostly interested in, among other more fine grained details about the query.
Parameters:
Name Type Description
initialDate Date Date to start counting the time period.
delay Number How many business days to add to the `initialDate`.
